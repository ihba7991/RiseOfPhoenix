# phoenix-b

Personal learning space. Rename this folder to your handle when ready (`git mv phoenix-b <handle>` and update `PEOPLE` in `docs/index.html`).

Layout mirrors the repo root:

- `ai/` — `build-your-own-ai.html` tracker
- `competitive-programming/`
- `hld/`
- `lld/`
- `languages/`
  - `build-your-own-backend-frontend.html`
  - `fafo-getting-your-hands-dirty.html`
  - `go/100_go_tasks.md`
  - `java/100_java_tasks.md`
  - `python/100_python_advanced_tasks.md`
  - `javascript/100_react_tasks.md`, `100_nodejs_tasks.md`

The HTML trackers save your checkbox state to `localStorage` — it lives on whichever browser you tick it from. The dashboard at `docs/index.html` shows recent commit activity across the repo.
